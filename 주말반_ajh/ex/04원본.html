가상요소

  • 가상요소는 가상의 태그(요소,span)가 있는것 처럼 사용
  • 내용(콘텐츠)을 작성x, 디자인용으로만 사용
  • content 속성은 필수!

CSS 선택자

  • id(#), class(.), tag(요소)
  • 가상선택자:
    • :nth-child(숫자) : (숫자)번째 형제요소로 선택
    • :nth-of-type(숫자) : (숫자)번째 같은 요소번호로 선택
    • :first-child : 첫번째 요소
    • :last-child : 마지막째 요소
    • :link/:visited/:active/:hover : 링크속성/특정상황
    • :focus : 활성화 되었을때
    • :checked/:selected : 체크가 되었을 때, 선택이 되었을 때
    • :empty : 비어있는 태그 일때
  • > : 자식선택자
  • (빈칸) : 자손선택자
  • [type=password] : 속성선택자
  • ~: 형제선택자(동생들)
  • + : 앞에 형제를 지정된 경우의 요소선택
  • , : 그룹선택자
